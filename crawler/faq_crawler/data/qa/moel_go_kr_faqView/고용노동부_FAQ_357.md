# 사업주직업능력개발훈련지원

## 카테고리

고용복지

## 답변

사업주가 소속근로자, 채용예정자, 구직자 등의 직무능력향상을 위하여 직업훈련을 실시한 경우, 이에 소용되는 비용의 일부를 지원하고 있습니다.
지원절차 및 기타 자세한 사항은 아래 내용을 참고해주시기 바랍니다.
o 지원 절차
- 훈련과정인정신청 → 훈련실시신고 및 훈련실시 → 훈련수료보고 → 비용신청
o 문의처
- 사업주훈련 일반문의: 사업장 관할 한국산업인력공단 지역본부·지사 (
https://hrdkorea.or.kr
)
- 사업주훈련 비용신청·조회: 고용24 (
https://www.work24.go.kr
), 한국산업인력공단 (
https://hrdc.hrdkorea.or.kr/hrdc/193483r
), 1644-8000(고객센터)
